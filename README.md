 # SpaceShip-Unity
Первый проект в Unity
## Теория
Данный проект представляет собой простую игровую сцену с городом внутри скалы и управляемым персонажем.
## Управление
W - движение вперед  
S - движение назад  
A - поворот влево  
D - поворот вправо  
Space - прыжок
Shift - бег
