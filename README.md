 # SpaceShip-Unity
Первый проект в Unity
## Теория
Данный проект представляет собой простую игровую сцену с управляемым короблем и планетой.
## Управление
W - движение вперед  
S - движение назад  
A - поворот влево  
D - поворот вправо  
Space - прыжок
Shift - бег
